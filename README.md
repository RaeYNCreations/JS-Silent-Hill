# JS' Silent Hill - NeoForge Mod

A NeoForge 1.21.1 mod that brings the terrifying creatures of Silent Hill into Minecraft.

## Features

### Entities
- **Pyramid Head** - The iconic executioner from Silent Hill 2. High health, devastating damage, and complete knockback resistance.
- **Mannequin** - Unsettling enemy with eerie sounds and moderate combat abilities.
- **Lying Figure** - Crawling monstrosity from Silent Hill 2.
- **Crying Girl** - Haunting enemy with disturbing presence.
- **Straightjacket** - Twisted enemy in a confining jacket.

### Items
- **Silent Hill Axe** - Custom weapon with high durability and damage.

### Sounds
Custom sound effects ported from the original Bedrock addon for an authentic horror atmosphere.

## Installation

1. Install [NeoForge 1.21.1](https://neoforged.net/)
2. Download the latest release of this mod
3. Place the mod JAR file in your `mods` folder
4. Launch Minecraft with the NeoForge profile

## Building from Source

```bash
./gradlew build
```

The built JAR will be in `build/libs/`
